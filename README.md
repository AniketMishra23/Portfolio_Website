# Aniket Mishra — Portfolio

> 🔗 **Live at [anisport.netlify.app](https://anisport.netlify.app)**

Personal portfolio website showcasing my projects, skills, and experience as a Data Scientist and ML Engineer. Fully mobile-responsive with a dark cyberpunk aesthetic.

---

## Built With

- **React 19 + Vite** — component-based UI with fast builds
- **Tailwind CSS v4** — utility-first styling, responsive at every breakpoint
- **Framer Motion** — scroll animations and transitions
- **react-type-animation** — typewriter effect in the hero section

---

## Sections

- **Hero** — animated intro with typewriter role cycling and resume download
- **About** — background, education summary, and personal traits
- **Projects** — AcadBot, AusFlash, IndianLeftOver, YouTube Q&A Bot, EHR System, CarPool — each with GitHub and live links
- **Skills** — animated proficiency bars, full tech stack, and certifications
- **Experience** — internships at LegalBrain.ai, Sahu Technologies, CodeClause
- **Education** — Master of Data Science (Deakin University) + B.E. ECE (TCET, Mumbai)
- **Contact** — contact form + links to GitHub, LinkedIn, and email
